# Hackintosh BigSur Asus X454L

<img src="https://i.imgur.com/F7DAQsQ.png" />

## Specs
CPU : I3-5005U  
GPU : Intel HD5500  
RAM : 8 GB  
OS : macOS Big Sur  
EFI : OpenCore 0.6.3  

## What works?
* Graphics HD5500 IGPU
* Sound
* HDMI Port
* Ethernet
* Keyboard
* Touchpad
* Battery
* Sleep
* Brightness

## What don't works?
* Nvidia 920M (disabled)
* VGA Port
* Card reader
* Wifi : Use Wireless USB Adapter
* Bluetooth : Use Bluetotth USB Adapter
